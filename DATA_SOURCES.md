# DATA SOURCE VERIFICATION
All data used in the application - verified sources only

## ACTIVE DATA FILES (Currently Imported and Used)

### 1. Solar suitability data
**Source:** Backend database (`solar_suitability` table)

**Real Data Sources:**
- **Land Cover:** NLCD 2024 Land Cover Raster
  - File: `Annual_NLCD_LndCov_2024_CU_C1V1.tif`
  - Source: USGS National Land Cover Database 2024
  - Resolution: 30m pixels
  - Data: Actual satellite-derived land cover classifications (15 NLCD classes)
  
- **Slope:** LandFire 2020 Slope Raster  
  - File: `LF2020_SlpD_220_CONUS/Tif/LC20_SlpD_220.tif`
  - Source: USGS LandFire 2020
  - Resolution: 30m pixels
  - Data: Actual terrain slope percentages from DEM

- **Population Density:** GPW v4 2020 Population Density Raster
  - File: `gpw_v4_population_density_rev11_2020_1_deg.tif`
  - Source: NASA SEDAC Gridded Population of the World v4
  - Resolution: ~1 degree (~100km)
  - Data: Actual population density (people per sq km)

- **Substations:** Michigan Power Facilities Database
  - File: `src/data/michigan_facilities.json` (extracted from Power_Plants_20250824_025710_chunk0000.geojson.gz)
  - Source: EIA Power Plants Database
  - Count: 722 facilities in Michigan
  - Data: Real geographic coordinates of power facilities

**Methodology:** EGLE Solar Suitability Scoring
- Land Cover weight: 4
- Substation Distance weight: 3  
- Slope weight: 2
- Population weight: 1
- Overall score = (sum of weighted scores) / 10

**Grid Coverage:** Stored in the backend database and queried by API

**STATUS:** ✅ 100% REAL DATA - No estimates or fake data
